# User Guide

## Accessing the App

1. Visit `http://localhost/STDDB`
2. You will see the **Harvard Student Database** table.

## Features

### Add New Student
- Click the "Add New Student" button.
- Fill in name, email, and course.
- Submit to add a record.

### Edit Student
- Click the "Edit" button next to any record.
- Update the fields and submit.

### Delete Student
- Click the "Delete" button.
- Confirm deletion.
